# Oris Strategy SA - Landing Page

A modern, fully responsive landing page built with HTML, Tailwind CSS, and JavaScript.

## Features

- **Fully Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Beautiful gradient design with smooth animations
- **Interactive Elements**: 
  - Smooth scrolling navigation
  - Mobile-friendly hamburger menu
  - Animated statistics counters
  - Contact form with validation
  - Back-to-top button
- **Sections Included**:
  - Hero section with compelling call-to-action
  - Services showcase (6 main services)
  - About section
  - Contact form with validation
  - Professional footer

## Technologies Used

- **HTML5**: Semantic markup
- **Tailwind CSS**: Utility-first CSS framework (via CDN)
- **JavaScript**: Vanilla JS for interactivity
- **Font Awesome**: Icons library
- **Google Fonts**: Inter font family

## Getting Started

1. Simply open `index.html` in your web browser
2. No build process or installation required - all dependencies are loaded via CDN

## Customization

### Colors
The main gradient colors can be changed in the CSS:
- Primary: `#667eea` (purple)
- Secondary: `#764ba2` (indigo)

### Content
- Edit the HTML file to update text content
- Modify service cards in the Services section
- Update contact information in the Contact section

### Form Submission
Currently, the contact form shows a success message. To make it functional:
1. Set up a backend API endpoint
2. Update the fetch request in `script.js` (commented code provided)
3. Handle form submission on your server
